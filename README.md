Overview

Glicontrol is a software application designed to support the management and monitoring of patient glucose levels.
It allows recording, storing, and analyzing measurements, providing a structured way to track patient data over time.

Key Features

Record glucose measurements with timestamp and contextual data (e.g., meals)
Store and manage patient data using a relational database
Visualize and analyze glucose trends
Structured and modular architecture for maintainability

Architecture

The application follows a layered architecture:

DAO Layer → data access and persistence
Business Logic Layer → data processing and validation
Presentation Layer → user interaction

This separation improves scalability, maintainability, and testability.
